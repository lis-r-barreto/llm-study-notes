# llm-study-notes

Study notes for generative AI, with a focus on large language models (LLMs) and AI agents.

## What This Repository Contains

This repository is organized as a personal learning space for notebooks, exercises, and reference material collected while studying generative AI courses.

## Repository Structure

- `001-hugging-face-llm-course/`: notes and notebooks from the Hugging Face LLM Course.
- `002-hugging-face-agents-course/`: notes and notebooks from the Hugging Face Agents Course.
- `003-zoomcamp-llm-course/`: notes and notebooks from the DataTalks Club LLM Zoomcamp.

## Courses

- [Hugging Face LLM Course](https://huggingface.co/learn/llm-course/)
- [Hugging Face Agents Course](https://huggingface.co/learn/agents-course/)
- [DataTalks Club LLM Zoomcamp 2026](https://courses.datatalks.club/llm-zoomcamp-2026/)
